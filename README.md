# ommurugaconstructions
Om Muruga Construction Website
Overview

This is a responsive and modern website for Om Muruga Construction, a professional construction company based in Tamil Nadu, India. The website showcases the companyâ€™s services, projects, and contact information with interactive features like a hero background slider, project image slider, and dark/light theme toggle.

All files (HTML, CSS, JS, images) are kept in the same folder for simplicity and easy deployment.

Features

Fully responsive design using Bootstrap 5.

Hero background slider with automatic image rotation.

Project image slider with previous/next controls.

Dark/Light mode toggle.

Sections included:

About Company

About Owner

Services

Projects

Contact

Interactive contact icons: Call, Email, WhatsApp, and Google Maps.

Floating social media icons: Instagram, Facebook, WhatsApp.

File Structure

All files are in a single folder:

index.html        -> Main HTML file
style.css         -> Stylesheet
script.js         -> JavaScript (sliders & theme toggle)
1.jpg, 2.jpg ...  -> Hero background images
img4 (1).jpeg ... -> Project images
logo.png          -> Company logo
Owner.jpeg        -> Owner image
10.png            -> Additional hero image

Deployment

The website can be hosted on Netlify or GitHub Pages.

Simply upload all files to your hosting platform; no folder structure is required.

Works seamlessly on desktop and mobile devices.

Usage

Clone or download this repository.

Ensure all images and files are in the same folder as index.html.

Open index.html in a web browser to view the website.

Customization

Replace the images (1.jpg, 2.jpg, etc.) with your own images to update the hero slider.

Update the projectImages in script.js to match your project images.

Modify text content in index.html to reflect your company details.
